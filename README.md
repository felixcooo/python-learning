# Python Fundamentals on Raspberry Pi OS

## Description
This repository contains beginner-friendly Python exercises and examples
developed on Raspberry Pi OS for educational purposes.
It covers the core fundamentals of Python programming,
designed for anyone starting their coding journey
on a Raspberry Pi device.

## Topics Covered
- Variables and data types
- Basic input/output (print, input)
- Arithmetic operators
- Strings and string methods
- Conditional statements (if, elif, else)
- Loops (for, while)
- Functions
- Lists and dictionaries

## Environment
- Device: Raspberry Pi 5
- OS: Raspberry Pi OS
- Language: Python 3

## Purpose
This project was created for educational purposes as part of
a personal Python learning journey. All examples and exercises
are beginner-friendly and well-commented for easy understanding.

## How to Run
1. Make sure Python 3 is installed on your Raspberry Pi
2. Clone this repository:
   git clone https://github.com/YourUsername/YourRepo.git
3. Navigate to the project folder:
   cd YourRepo
4. Run any Python file with:
   python3 filename.py
